## 🚀 **Getting Started**

### **Prerequisites**
- Flutter SDK 3.41.4+
- Node.js 24.11.1+
- Android Studio / VS Code
- Git

### **Installation**

1. **Clone Repository**
   ```bash
   git clone [repository-url]
   cd flutter_application_1
   ```

2. **Install Dependencies**
   ```bash
   flutter pub get
   npm install  # for backend
   ```

3. **Start Backend Server**
   ```bash
   cd ../backend
   node server.js
   ```

4. **Run Flutter App**
   ```bash
   flutter run -d chrome --web-port=3000
   ```
